
A web application that connects hotels with NGOs to reduce food waste.
Developed by
Sancta Fannie

 Features
- Hotel registration & login
- NGO registration & login
- Post surplus food
- NGOs can view & request food
- Dashboard for hotels and NGOs

 Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS
- GitHub
